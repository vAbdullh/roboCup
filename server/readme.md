# RoboCup

## Match Endpoint

### GET /match
Retrieve all matches.

### POST /match
Create a new match.

### POST /match/end/:matchId
End a match.

### POST /match/start/:matchId
Start a match.

### POST /match/break/:matchId
Set a break for a match.

### PUT /match/:matchId
Update a match by ID.

### DELETE /match/:matchId
Delete a match by ID.
